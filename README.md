# Java To-Do List Application

## 📌 Description
This is a console-based To-Do List application developed in Java.  
It allows users to manage tasks by adding, viewing, and removing them through a simple menu system.

## 🚀 Features
- Add new tasks
- View all tasks
- Remove tasks
- Mark tasks as completed
- Menu-based user interaction

## 🛠️ Technologies Used
- Java
- ArrayList
- Scanner

## 🧠 Concepts Applied
- Object-Oriented Programming (OOP)
- Data Structures (ArrayList)
- State management (task completion tracking)
- Control Flow (loops, switch statements)
- User Input Handling

## ▶️ How to Run
1. Open the project in a Java IDE (e.g. IntelliJ or Eclipse)
2. Run the `ToDoList` class

## 📈 Future Improvements
- Save tasks to a file
- Add graphical user interface (GUI)
